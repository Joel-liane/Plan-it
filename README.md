# Plan It

A web-based todo list application to manage tasks efficiently, built with PHP, MySQL, and JavaScript.

## Features
- User registration and login with password hashing
- CSRF protection for secure forms
- Create, edit, delete, and mark tasks as done/pending
- Task due dates with overdue and urgent indicators
- Task filtering (all, pending, done)
- Session management and logout
- AJAX-powered task completion toggle

## Tech Stack
- PHP
- MySQL (MariaDB)
- HTML / CSS / JavaScript
- Git for version control

## Installation

1. Clone the repository:
   git clone https://github.com/Joel-liane/plan-it.git
2. Move to project folder:
   cd plan-it
3. Make sure you have XAMPP or another PHP server installed
4. Import `plan_it.sql` into your MySQL database
5. Update `db.php` with your database credentials
6. Open in browser: http://localhost/plan-it/
